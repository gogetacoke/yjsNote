# tomcat 启动失败

**原因**：缺少一个生成随机字符的程序

**解决办法**：

```
ln -s /dev/urandom /dev/random  # 制作一个软连接

# 没有urandom
yum -y install rng-tools
systemctl start rngd
systemctl enable rngd
```

启动成功标识：

```
ss -ntulp|grep java # 查询启用了三个端口 8005 8009 8080 即启动成功
```
