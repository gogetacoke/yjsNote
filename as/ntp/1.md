# 安包

```
yum -y install chrony
```

# 时间服务器配置

```
vim /etc/chrony.conf
# 第三行注释
# pool 2.pool.ntp.org iburst 
# 22行修改
allow all # 允许所有IP访问
# 25行打开注释
local stratum 10 # 使用第10层时间服务器

systemct start chronyd
```
